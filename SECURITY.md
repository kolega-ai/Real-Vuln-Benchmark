# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

**Do NOT open a public GitHub issue for security vulnerabilities.**

Instead, please email us at: **security@kolega.dev**

We will acknowledge your report within 48 hours and aim to provide a fix or mitigation within 7 days.

## Scope

This repository contains a benchmark for evaluating security scanners. The ground-truth data and scan results describe vulnerabilities in *other* intentionally-vulnerable repositories — not in this project itself.

If you find an issue with the benchmark tooling (e.g., the scoring pipeline, dashboard, or parsers), please report it through the standard issue tracker.
