\section{Data Availability}

All artifacts described in this paper are publicly available under open licenses.

\begin{description}
  \item[Benchmark code (MIT License).]
    Scoring pipeline (\texttt{score.py}, \texttt{scorer/}), parser library
    (\texttt{parsers/}), and interactive dashboard generator
    (\texttt{dashboard.py}).
    \url{https://github.com/kolega-ai/RealVulnBenchmark}

  \item[Ground truth.]
    Hand-labeled vulnerability annotations for all 26 repositories, stored as
    validated JSON in \texttt{ground-truth/\{repo\}/ground-truth.json}.
    \url{https://github.com/kolega-ai/RealVulnBenchmark}

  \item[Scan results.]
    Raw output from all 15 evaluated scanners across every repository, stored
    in \texttt{scan-results/\{repo\}/\{scanner\}/results.json}.
    \url{https://github.com/kolega-ai/RealVulnBenchmark}

  \item[LLM evaluation harness.]
    Prompt templates, single-turn and agentic runner scripts, and per-run cost
    tracker used to produce LLM scanner results.
    \url{https://github.com/kolega-ai/RealVulnBenchmark}

  \item[Live dashboard.]
    Interactive results explorer with per-scanner, per-repository, and
    per-CWE-family breakdowns, rebuilt automatically on each benchmark release.
    \url{https://kolega-ai.github.io/RealVulnBenchmark}

  \item[Benchmark manifest.]
    Version-locked reproducibility file (\texttt{benchmark-manifest.json})
    recording ground-truth content hashes, upstream repository commit SHAs,
    and prompt versions, ensuring exact score reproducibility.
    \url{https://github.com/kolega-ai/RealVulnBenchmark}

  \item[Paper.]
    Preprint available on arXiv; camera-ready version linked from the
    repository README.
\end{description}

\noindent All of the above are accessible at:
\url{https://github.com/kolega-ai/RealVulnBenchmark}
